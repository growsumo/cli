GrowSumo CLI is now in v1.0!

The fastest blockchain backed, anonymous, easy to use, command line referral platform

We have completely decentralized the referral process by integrating directly with the GrowSumo chain. A CLI interface intruments a gateway to the most durable distributed referral platform.

Here are some of the new features released with v1.0

## Connect your GTC wallet

This can take up to 12 days to sync with the current state of the chain 
$ gs wallet -n e3ndn92@5%d1ja22901m@.wallet

## Submit a lead's IP address

$ gs send -i 158.0.2.443

## List your referrals

$ gs referrals

3F4C44F6021B0A  [approved]
C0E7F2C3E3E32B  [pending]
59C17E8E83A69E  [declined]
25932C2325CAFD  [approved]

## View a graph of your performance over time

$ gs graph -t line -t1 03:31:2018:00:00:00 -t2 04:01:2018:23:59:59
